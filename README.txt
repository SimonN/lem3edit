Lemmings 3 Level Editor (lem3edit SVN)
================================================================================

syntax: lem3edit <level_number>

place the binary in the same directory as L3CD.EXE
on case-sensitive filesystems, all the data files must have upper case filenames

== Keys ========================================================================
s             -- save level
q             -- quit

a             -- select all
ctrl          -- select multiple
escape        -- select none
delete        -- delete selected objects
ctrl + c      -- copy
ctrl + v      -- paste
z             -- decrease object ID
x             -- increase object ID

scroll-wheel  -- zoom in and out
b             -- toggle background-only mode
ijkl          -- scroll
arrow pad     -- move objects
shift         -- ... faster
